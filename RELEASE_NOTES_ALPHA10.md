# VyoraXR 0.1.0-alpha10

## Meta Store patch notes

- Verbeterde VR-controllerafhandeling voor grip, trigger en B.
- VR-zoom blijft beschikbaar via up/down op de joystick.
- Horizontale joystickbesturing voor het verplaatsen van beeld is uitgeschakeld; verplaatsen gebeurt via grip.
- Stabiliteits- en player-inputverbeteringen.

## Test note

Deze alpha bevat een experimentele wijziging aan de Quest-controllerinput in VR playback.
